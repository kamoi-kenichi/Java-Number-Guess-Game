# Java 数当てゲーム

## 概要
Javaで制作したコンソール型の数当てゲームです。  
難易度選択、ヒント表示、入力バリデーションに加え、  
ベストスコア保存・プレイ回数記録・最終プレイ日時保存を実装しています。

シンプルなゲームを題材に、状態管理とデータ永続化を意識して設計しました。

## 制作目的
Javaの基礎文法を活用しながら、条件分岐・繰り返し処理・Map管理・ファイル保存などを実践的に学ぶことを目的に制作しました。

## 主な機能
- 難易度選択
- 数値入力バリデーション
- ヒント表示
- 難易度別ベストスコア管理
- プレイ回数の記録
- 最終プレイ日時の保存
- テキストファイルによるデータ永続化

## 使用技術
- Java
- HashMap
- Scanner
- Random
- PrintWriter
- LocalDateTime
- DateTimeFormatter

## 工夫した点
- Mapを使って難易度ごとのスコアを管理  
  → 条件分岐を減らし、拡張しやすい構造にしました
- 保存データの読み込み時に形式チェックを実装  
  → 外部データをそのまま信用しない設計を意識しました
- 不正データがあっても処理が停止しないよう例外処理を追加  
  → アプリ全体の安定性を優先しました
- プレイヤーが遊びやすいよう、提示した数との差に応じたヒントを表示  
  → 単調にならないゲーム体験を意識しました

## 実行方法
```bash
javac Main.java
java Main
```

## データ保存について
ゲームデータは `game_data.txt` に保存されます。

保存内容：
- プレイ回数
- 最終プレイ日時
- 難易度別ベストスコア
